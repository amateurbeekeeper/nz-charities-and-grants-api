NZ Charities API

Free, open, static API for all 45,000+ registered charities in New Zealand.
Data sourced from Charities Services NZ.

Endpoints

All charities

GET /api/charities.json

Single charity by registration number

GET /api/charities/{CC_NUMBER}.json

# Example
GET /api/charities/CC10001.json

Index by city

GET /api/cities.json

Registration statuses

GET /api/statuses.json
